# 🎬 Next.js Movie Browser

Bu proje, OMDb API ile entegre çalışan, TypeScript + Next.js 15 + Tailwind CSS kullanılarak geliştirilmiş bir film arama ve listeleme uygulamasıdır.

## 🔍 Özellikler

- Film arama (varsayılan: "batman")
- Poster, isim, yıl bilgisi ile film kartları
- Detay sayfasında:
  - Özet
  - Yönetmen
  - Tür
  - IMDb puanı
  - Süre
- Responsive tasarım (mobil uyumlu)
- Server Component kullanımı (Next.js 15)

## 🧱 Teknolojiler

- [Next.js 15](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [OMDb API](https://www.omdbapi.com/)
